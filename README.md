#  AURA FITNESS SYSTEM

AURA FITNESS is a fitness transformation platform designed to showcase structured 90-day training programs and guide users through a disciplined fitness journey. It provides a clean, modern browsing experience focused on fitness goals, consistency, and progression.

---

# Overview

The website is designed to create a motivating and structured online experience for users interested in fitness transformation, strength building, endurance, and overall body development.

---

# Visual Identity

- **Images:** High-quality fitness training images showing workouts, gym environments, transformation journeys, and active lifestyles  
- **Typography:** Bold, modern sans-serif fonts that reflect energy, discipline, and motivation  
- **Color Theme:** Dark modern fitness palette using black, red, grey, and subtle white accents
